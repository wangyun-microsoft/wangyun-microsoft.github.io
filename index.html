Testing page: Build Word Writing Assistant Add-in

In this article, you'll walk through the process of building a Word task pane add-in with Teams Toolkit.

Prerequisites

  • Node.js (the latest LTS version). Visit the Node.js site to download and install the right version for your operating system.

  • Visual Studio Code

  • Please make sure you've downloaded and installed the testing version of Visual Studio Code extension in the call for email you've received. Once we go to GA, this install process will be automated.

  • Please watch our two-min video here before moving forward. You will understand what buttons to click inside Visual Studio.

Trouble shooting

Sometimes your VS Code setting is to automatically upgrade extensions to the latest version. You need to make sure your installed version is v5.4.1.

You need to make sure your Word application is closed before running the add-in sample. If you have multiple Word document open, you may meet 'add-in not loaded' problem.
